    模型

    已接入模型

    当前还没有已接入模型。

    内置订阅型 Provider 可以直接选预设；如果是 Pi 已存在的 provider，也可以直接填写上面的 Provider Key。

    桌面版将优先通过 Pi CLI `/login` 完成登录；当前 Web 版可先使用“手动导入 token”作为兼容方案。

    支持直接粘贴主流厂商使用的整段 JSON 配置。可以传 { "models": [...] }、直接传模型数组 [...]，也可以粘带有 baseUrl、apiKey、api 的完整对象，系统会自动提取并回填上方字段。API 模式下至少需要配置一个模型。

    OAuth 登录会打开终端中的 Pi agent。打开后请在终端里手动输入 /login，再由你自己选择支持的模型提供商并完成登录。
